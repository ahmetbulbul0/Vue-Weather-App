# Vue-Weather-App

***Vue Weather App Project***

## Technologies
* **Html**
* **Css**
* **Js**
* **Vue**

## Project Setup Process

### install npm (for javascript packages)
```sh
npm install
```
### to run project in development environment
```sh
npm run dev
```

### Compile and Minify for Production
```sh
npm run build
```

### Lint with
```sh
npm run lint
```

## Dependencies
* Vue
* Vue-Router
